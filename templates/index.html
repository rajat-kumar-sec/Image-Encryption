ℹ️ About This Tool

🛡️ Security Features

  • AES-256 Encryption: Military-grade symmetric encryption
  • Fernet (HMAC+AES): Authenticated encryption for data integrity
  • SHA-256 Key Derivation: Secure password-to-key conversion
  • Client-Side Processing: All encryption happens locally - no uploads
  • Zero Knowledge: Your data never leaves your browser

🔧 How It Works

  1. Upload your image file
  2. Enter a strong password
  3. Click "Encrypt" - image is encrypted using AES-256
  4. Download the encrypted .enc file (unreadable without password)
  5. To decrypt, upload the .enc file and enter the same password

🔐 Password Tips

  • Use at least 12 characters
  • Mix uppercase, lowercase, numbers, and symbols
  • Avoid common words or personal information
  • Use unique passwords for different images
  • Store passwords securely

📊 Technical Details

Encryption Method: Fernet (symmetric)
Key Length: 256-bit
Algorithm: AES (CBC mode)
Authentication: HMAC-SHA256
Data Location: Client-side only

⚠️ Important Notes

  • Keep your password safe - without it, data cannot be recovered
  • This tool uses open-source cryptography libraries
  • No server-side storage - refresh the page and your data is gone
  • For maximum security, run this tool on a trusted device
